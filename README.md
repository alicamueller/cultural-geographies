# cultural-geographies

Five spatial analyses on culture, memory, and urban space in Germany.

Each project applies a different GIS method to a question about 
how space shapes — and is shaped by — cultural life. Taken together, 
they ask: whose culture is visible, accessible, and remembered 
in the city?

GIS workflows are underrepresented on GitHub. These projects are 
documented to be fully reproducible — each README includes 
step-by-step methods.

---
## Projects

**viewshed-cologne-cathedral**  
Which cultural institutions are visible from Cologne Cathedral?  
*Tools: QGIS · GRASS r.viewshed · QuickOSM*

**streets-named-after-artists-leipzig**  
Which districts of Leipzig have the highest concentration of streets 
named after artists?  
*Tools: QGIS · Excel*

**cultural-accessibility-dresden**  
Does political geography correlate with cultural accessibility?  
*Tools: QGIS · QNEAT3*

**site-specific-performance-rostock**  
Which park areas in Rostock are suitable for a site-specific performance?  
*Tools: QGIS*

**tree-cover-change-hellerau**  
How has tree cover changed in Germany's first garden city between 
2005 and 2022?  
*Tools: QGIS · YOLOv9 · Python*

## Stack
QGIS · GRASS GIS · QNEAT3 · YOLOv9 · Python · OpenStreetMap
